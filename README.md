Product Management System - Python Final Project
🇪🇸 Descripción (Español)
Este proyecto es un Sistema de Gestión de Inventario basado en consola, desarrollado como trabajo final para el curso de QA Tester. El objetivo principal fue aplicar lógica de programación en Python para resolver un problema de gestión de datos, haciendo especial énfasis en la validación de entradas y el manejo de errores de usuario.
✨ Características:
Gestión CRUD completa: Permite agregar, listar, buscar y eliminar productos.
Validación de Datos (QA Focus):
Control de campos obligatorios (evita ingresos vacíos).
Validación de tipos de datos (asegura que el precio sea un número entero).
Reglas de negocio (el precio debe ser un número positivo).
Interfaz de Usuario: Menú interactivo por consola con confirmaciones de seguridad antes de eliminar datos.
🇺🇸 Description (English)
This project is a console-based Inventory Management System developed as a final project for a QA Tester course. The main goal was to apply Python programming logic to data management, with a strong focus on input validation and user error handling.
✨ Features:
Full CRUD Management: Allows adding, listing, searching, and deleting products.
Data Validation (QA Focus):
Mandatory field control (prevents empty inputs).
Data type validation (ensures price is an integer).
Business rules (price must be a positive number).
User Interface: Interactive console menu with safety confirmations before deleting data.
🛠️ Tech Stack
Language: Python 3.10+
Concepts: Lists, Loops, Conditionals, and String Sanitization.
