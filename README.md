# Product Management System - Python Final Project

## 🇪🇸 Descripción (Español)
Este proyecto es un **Sistema de Gestión de Inventario** basado en consola, desarrollado como trabajo final para el curso de **QA Tester**.

### ✨ Características:
* **Gestión CRUD completa:** Permite agregar, listar, buscar y eliminar productos.
* **Validación de Datos (QA Focus):** Control de campos obligatorios y tipos de datos.
* **Interfaz de Usuario:** Menú interactivo por consola.

---

## 🇺🇸 Description (English)
This project is a console-based **Inventory Management System** developed as a final project for a **QA Tester** course.

### ✨ Features:
* **Full CRUD Management:** Allows adding, listing, searching, and deleting products.
* **Data Validation (QA Focus):** Mandatory field control and data type validation.
* **User Interface:** Interactive console menu.

---

### 🛠️ Tech Stack
* **Language:** Python 3.10+
